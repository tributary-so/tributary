# Tributary

Quick start guide for running the Tributary application.
Tributary - a river or stream that flows into a larget river or a lake

## Prerequisites

- Node.js 20.19+ or 22.12+
- pnpm 9.6.0+

## Setup & Run

gh repo clone xeroc/tributary
cd tributary
pnpm install

cd sdk
pnpm build

cd ../ssdk-react
pnpm build

cd ../app
pnpm run dev
